Создание функции по графу

Функция описывается следующим образом:
Если у вершины с именем «A» нет ни одного входа, то функция вершины равна
её имени. Можно также после имени поставить скобки.
Пример: A
Если в вершину с именем A входит N > 0 входных дуг из вершин с именами A1, …, Ak
(порядок задаётся номерами в списке входных вершин), то функция вершины
записывается в префиксном виде с параметрами, равными функции от вершин
FA1, …, FAk:
Пример: A(FA1, …, FAk)
